## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ManuLisek/to-do-list-react.git
   ```
2. Open the root folder
   ```sh
   cd to-do-list-react
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run project
   ```sh
   npm start
   ```
